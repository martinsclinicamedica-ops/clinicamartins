# Clínica Martins — versão 3

## Publicação no GitHub Pages

1. Apague todos os arquivos antigos do repositório.
2. Extraia este ZIP no computador.
3. Abra o repositório no GitHub.
4. Clique em **Add file → Upload files**.
5. Envie todos os arquivos e pastas desta versão de uma única vez.
6. Faça o commit.
7. Em **Settings → Pages**, confirme que a fonte é `main` e `/root`.
8. O arquivo `CNAME` já configura o domínio `clinicamartins.med.br`.

Não misture estes arquivos com versões anteriores.
